## Sobre o projeto

O portfólio foi desenvolvido com estrutura modular, separando páginas e estilos para melhor organização e manutenção do código.

O foco foi criar uma interface simples, clara e funcional, com navegação intuitiva e boa adaptação a diferentes tamanhos de tela.

---

## Funcionalidades

- Navegação entre páginas (Sobre mim, Formação, Experiências, Contato);
- Layout responsivo (adaptado para diferentes telas);
- Menu lateral fixo;
- Links externos para redes profissionais;
- Botão para envio de e-mail;
- Feedback visual com interações em CSS e JavaScript.
